# Data Folder

This folder contains all data used in the project.

## Subfolders:
- **simulations/**: Raw data generated by simulations.
- **experimental/**: Raw data collected from real-world experiments.
- **processed/**: Cleaned and processed data ready for analysis.
- **tmp/**: Temporary or intermediate data files *(ignored by version control)*.

## Usage Tips:
1. **Immutability**: Do not modify raw data in `simulations/` or `experimental/`.
2. **Processing**: Store processed or aggregated data in `processed/` with clear labeling.
3. **Temporary Files**: Use `tmp/` for intermediate files but clean it regularly to save space.
